# DocIDV Sample App

A sample iOS application demonstrating the basic integration of the IDnow DocIDV SDK.

## Getting Started

Clone the repository and open it in your IDE:
```bash
git clone https://github.com/idnow/docidv-sdk-ios-sample.git
```

---

## What's Demonstrated
- Starting an identification session with **IDnowDocIDV**
- Handling session results

---

## Dependencies

This sample uses the IDnow DocIDV SDK. See the [official documentation](https://www.idnow.io/developers/) for integration details.
